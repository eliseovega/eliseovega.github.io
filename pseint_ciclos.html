1. Introducción

Que es un Ciclo o Bucle ?

Los ciclos ó bucles son una forma práctica de ejecutar el mismo código durante cierto número de veces o hasta que se cumpla una condición específica. Al realizar un bucle, el bloque de código que se ejecuta podría tener un valor diferente cada vez o permanecer igual.

Proposito

El propósito principal de un ciclo es iterar sobre un o un conjunto de bloques de codigo. La iteración es un acto común en el desarrollo de software. Repetir simplemente significa repetir una acción varias veces.

Sabemos que los bucles se utilizan en la programación para automatizar las diferentes tareas repetitivas.

Partes de un Ciclo o Bucle

En general, el ciclo o bucle se divide en 3 partes:

  1. Inicializacion: Asignación de un valor a la variable
  2. Condición: Condición para verificar si es verdadero o falso
  3. Incremento ó Decremento: se utilizan para incrementar automaticamente el ciclo o decrementar.

Tipos de Ciclos ó Bucles

Hay diferentes formas de recorrer e iterar:

  • Ciclo Mientras (White)
  • Ciclo Para (For)
  • Ciclo Repetir (Do White)

Nota Importante

Los ciclos ó bucles juegan un papel importante en la programación. A medida que ayudan a reducir las tareas repetitivas y autumatizarlas. Si tenemos la lógica correcta, el ciclo ahorra mucho tiempo en la programación.

Para ello es importante que analices las 3 parte de un ciclo y te ensucies las manos de codigo practicando su lógica.
