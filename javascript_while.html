EL CICLO WHILE

2. Sintaxis

El ciclo While ejecuta un bloque de código repetidamente mientras cierta condición es verdadera y deja de ejecutarla cuando la condición se vuelve falsa. La sintaxis se escribe, como se muestra a continuación:

El ciclo While primero analiza la condición, y si el resultado es verdaero procederá a ejecutar el bloque de instrucciones.

Nota Importante

Los ciclos ó bucles juegan un papel importante en la programación. A medida que ayudan a reducir las tareas repetitivas y autumatizarlas. Si tenemos la lógica correcta, el ciclo ahorra mucho tiempo en la programación.

Para ello es importante que analices las 3 parte de un ciclo y te ensucies las manos de codigo practicando su lógica.
